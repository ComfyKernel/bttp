BTTP
By ComfyKernel

BTTP : API v 0.1-Early

Name Description
bttp::init() Initializes basic BTTP classes
bttp::parameter Container class for parameter functions
bttp::parameter::get(const std::string&) Grabs a parameter from the pre compiled list
bttp::parameter::all() Returns the entire list of parameters
bttp::severity Enum for debug severity hints
bttp::getseverity(bttp::severity) Returns string related to severity
bttp::debug Container class for debug functions
bttp::debug::callback(void(*)(const bttp::severity, const std::string&, const std::string&)) Set callback for debugging functions
bttp::debug::callinfo/callwarn/callerror/callcrit/(const std::string&, const std::string&) Calls the set debug callback with one of the severities
bttp::logfile(const std::string&) Set log file
bttp::log Log used for writing to the file and printing to the screen
